# COVID-19 Data Immersion & Data Wrangling Project

##  Project Overview
This project focuses on **Data Immersion** and **Data Wrangling** using the global COVID-19 dataset. The objective is to explore, clean, transform, and analyze real-world COVID-19 data to generate meaningful insights through visualization and exploratory data analysis (EDA).
The project demonstrates how raw datasets can be converted into structured and analysis-ready data using Python and data analytics libraries.
## Objectives
- Understand dataset structure and quality
- Handle missing and inconsistent values
- Perform data cleaning and preprocessing
- Transform and organize data for analysis
- Analyze global COVID-19 trends
- Create meaningful visualizations and insights
